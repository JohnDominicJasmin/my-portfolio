Project Overview

Byahero is a lighter version of Google Maps but with Authentication and Live Weather Updates. Weather Updates help track weather forecasts and current conditions in real-time. You can view the upcoming 24-hour forecast.

Byahero is my first Android App. While building this app, I learned so many things like how to use APIs, Persistent Storage, Networking, Permissions, Architecture, Clean code practices, Material Designs, etc. I created this project for my family members to replace their Google Maps with a lighter version.
